# WUSM Assign DAG

A module to assign DAGs for multisite public surveys.
********************************************************************************

## Getting Started

Append the query string &dag=YOUR_DAG_UNIQUE_NAME
> Example:  https://redcapURL/surveys/index.php?s=84MY9NREFJ&dag=YOUR_DAG_UNIQUE_NAME
